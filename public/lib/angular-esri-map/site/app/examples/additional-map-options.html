Fade on Zoom: {{ map.mapOptions.fadeOnZoom }}, Logo: {{ map.mapOptions.logo }}, Wrap Around 180: {{ map.mapOptions.wrapAround180 }}

This example defines additional map options in the parent contoller's $scope.mapOptions.

Alternatively, map options can be supplied as an inline object hash string in the HTML template, such as map-options="{maxZoom: 10,...}".

These map options will also work alongside a webmap-id inline attribute.
